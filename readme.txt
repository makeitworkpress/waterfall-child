== Description ==

Waterfall Child is the child theme for Waterfall, which makes is very easy to build upon the Waterfall theme and develop advanced functionalities.

== Installation ==
The Waterfall child theme requires the [Waterfall Theme](https://github.com/makeitworkpress/waterfall).

1. Go to https://github.com/makeitworkpress/waterfall-child/
2. Download the zip package
3. Go to 'Appearance > Themes' and click 'Add New'
3. Click 'Upload Theme' on the top left
4. Select the downloaded them from your computer
5. Activate the child theme

== Privacy ==
Waterfall child does not use localStorage or any cookie mechanisms.
